# TrumpGame
### Javaプログラミング最終課題制作物.<br>
#### トランプで移動する宝探しゲーム
`javac Card.java Computer.java Deck.java DeckCheck.java Human.java Player.java Stage.java TrumpGame.java`<br>
から<br>
`java TrumpGame`で起動.<br>
<br>
- 勝利条件
  - P(Player)を操作してG(Goal)に到達する.(ゴールは北海道宗谷岬)
- 敗北条件
  - E(Enemy)と同マスに侵入する.
  - 山札・手札のトランプがなくなる.
